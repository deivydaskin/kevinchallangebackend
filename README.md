# kevinchallangebackend

# Installation

1. Clone repo

Using docker: 
2. `docker-compose up` 
3. If you haven't yet start front-end app - https://github.com/deivydaskin/kevinchallange

Directly: 
2. `npm install` 
3. `npm run dev` 
4. If you haven't yet start front-end app - https://github.com/deivydaskin/kevinchallange
